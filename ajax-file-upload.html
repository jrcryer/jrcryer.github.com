AJAX File Uploader

jQuery AJAX File Uploader

Simple jQuery plugin to make AJAX file uploading easy!

Example

            (function($) {
                $('form').ajaxFileUploader({
                    input: '.files',
                    hideButton: true,
                    onSuccess: function(text) {
                        $('.info').text(text);       
                    }
                });
            })(jQuery);

Options

input
Defaults to #files. Selector of file input form field
onSuccess
Callback when files are successfully uploaded.
onFailure
Callback when files fail to upload.
hideButton
Defaults to false. Hide submit button and automatically upload files as user selects them.
buttons
The buttons that will be hidden when hideButtons is set to true
