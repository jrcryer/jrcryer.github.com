Postez For WordPress

Addthis Analytics WordPress

WordPress widget plugin to display addthis analytics information about a blog.

Google Site Map for WordPress

Generates a Google Sitemap automatically for a WordPress site.

Sitemap is automatically refreshed when content is added, updated or removed.

Plugin adds an admin page allow an admin to modify content of the sitemap.
