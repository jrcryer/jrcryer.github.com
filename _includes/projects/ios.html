Notez

An experimental application to understand how CoreData integrates with UITableViewController.

CoreDataPref

An experimental application to perform a number of tests against CoreData.

The built application allows a user to run each test and displays the time taken to extract / update data via CoreData.

This application can be used to understand the best extraction strategies when using CoreData.
